# InfluenceOS Brand Guidelines

This document defines the brand identity, naming conventions, and visual standards for InfluenceOS.  
All contributors, partners, and integrators must follow these guidelines when referencing or representing the platform.

---

## 📛 Brand Name

| Usage                  | Correct Form       |
|------------------------|--------------------|
| Product name           | **InfluenceOS**    |
| Short form             | **IOS** *(internal only — avoid in public copy)* |
| Repository / package   | `influencer-os`    |
| CLI / code identifiers | `influenceos`      |

> ❌ Do not write: `Influence OS`, `influence_os`, `InfluencerOS`, or `Influencer OS`

---

## 🎯 Brand Mission

> **Empower creators, teams, and organizations with a modular, AI‑native operating system built for the influence economy.**

InfluenceOS exists to give creators enterprise‑grade infrastructure — the same tools used by large organizations, accessible, programmable, and fully automated.

## 🗣️ Brand Voice

| Attribute       | Description                                                  |
|-----------------|--------------------------------------------------------------|
| **Confident**   | Direct, authoritative, no hedging                            |
| **Technical**   | Precise language; uses correct terminology                   |

## 🧩 Logo Usage

- Always use the official InfluenceOS logo asset on a background that provides sufficient contrast
- Do not stretch, rotate, recolor, or add effects to the logo
- Maintain clear space equal to the height of the "I" in InfluenceOS on all sides
- Do not place the logo on busy or low-contrast backgrounds

---

## 🏷️ Product Module Naming

All modules follow the **one-word, lowercase** naming convention:

| Module Name     | Full Name                   |
|-----------------|-----------------------------|
| `engagement`    | Engagement Engine           |
| `contests`      | Contest Engine              |
| `resources`     | Resource Manager            |
| `security`      | Security Core               |
| `insurance`     | Insurance & Risk Engine     |
| `updates`       | Update Engine               |
| `ai`            | AI Layer                    |
| `autopilot`     | Autopilot Engine            |
| `marketplace`   | Marketplace Module *(future)* |
| `mobile`        | Mobile Layer *(future)*     |

> New modules **must** follow this convention. Multi-word module names are not permitted.

---

## 🤝 Partner & Integration Branding

When building integrations or partner products on top of InfluenceOS:

- You may state: *"Built on InfluenceOS"* or *"Powered by InfluenceOS"*
- You may use the InfluenceOS name in your product description with proper attribution
- You may **not** imply your product is an official InfluenceOS product without written permission
- You may **not** use the InfluenceOS logo as your own product logo
